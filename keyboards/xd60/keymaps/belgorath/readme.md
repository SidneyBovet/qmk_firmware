# Belgorath's Keymap for XIUDI's 60% XD60 PCB

![Layer 1](https://img.alicdn.com/imgextra/i1/1713761720/TB2K0gTalPxQeBjy1XcXXXHzVXa_!!1713761720.png)

## Additional Notes

This keymap for XD60 works best on Windows, as it provides virtual desktops navigation, media, and other stuff.

## Build

To build the default keymap, simply run:

    make xd60/rev2:belgorath  # XD60 rev2
    make xd60/rev3:belgorath  # XD60 rev3
